Splash Page

Technical Description

HTML Techniques: Used HTML techniques like meta refresh tag to redirect users to the Home page in 4 seconds.Used section and div tag to structure the content.Used h1 for the heading, p for the description, and ul to make a list of the group members for a proper information layout.Used img tag to enter the image of the logo to the splash screen. Implemented an animated loader within a div tag to provide a visual cue for the users while they wait.

CSS Techniques: Used CSS techniques to apply a full screen backgrounf image using (url('images/background.jpg') no-repeat center center/cover;) so the text of the page is written over the image.Used an .overlay with rgbs(0,0,0,0.7) to improve the text readability. Used properties of the flex bow like display: flex ;felx-direction: column ;justify-content: center; align-items: center; text-align: center; to align the content to the center.

Set a fixed position using position: fixed; width: 100%; height: 100% to keep the splash screen consistent. Created a rotating loader using with @keyframes spin , and created a border with 5px thickness and a white border-top and set the animation spin for 1 second linear infinite for a smooth rotation of the loader. Applied border-radius of 50% to give the image of the logo a rounder shape.

Accessibility

Using white text over a dark overlay improves the readability of the content.Ensured that images contain alt attributes ,so it can assist the users if by chance the image is not visible. Kept the animated loader simple with colours that makes it clear.

Volunteer Page

Technical Description

HTML Techniques: The content is wrapped in sections with an HTML section tag, which seperates each section with the class .volunteer-section for volunteer opportunities, .reviewsfor the reviews section and .feedback-form for getting user feedback.Within the sections div tags are used to further seperate different parts within the section. A select tag is used to creat dropdown list for users to sort the volunteer programs by popularity and recency or view all.The volunteer programs are presented as cards using div class="card-container" to ensure a grid like alignment.img tags are used to insert images to the volunteer-cards button tags are used to create 2 button, one for viewing more details and theother for applying. Used a textarea tag with sepcific number of rows and columns to ajust its size, which allows users to submit their suggestions and Action button are used to send feedback or cancel. A div tag is used with the id : "error-message" to display the error message.

CSS Techniques: Used padding and rounded borders in the main section for identification and design.Used a grid layout ( display: grid; grid-template-columns: repeat(2, 1fr);) for the volnteer card container.Used bow-shadow to show a soft shadow for depth.The width of the cards are calculated in way so that it fills the card container with padding. Used a flex-box for even spacing between the program details.Used object-fit: cover; along with 100% width and appropriate height so the images fit properly inside the card. Used a Hover effect on the buttons to make them pop when hovered.Used the color: red to clearly see the Error text when displayed and dispaly: none to initially hide the error message. Used properties like padding, magin, width, height,text-align, background-color to improve the outlook of the webpage.

JavaScript Techniques: Used addEventListener method to attach an event listener to the send-feedback element.Used event.preventDefault() method to prevent default behavior of the feedback submission.The display property is chaned from none to block to make the error message visible.

Accessibility

Ensured that images contain alt attributes ,so it can assist the users if by chance the image is not visible.Used a high contrast between text and background for readability.Used descriptive labels for dropdowns and form inputs for clarity.

Content Page

Technical Description

HTMLTechniques: The page is structured using section tags to logically group content.Header tags like h1, h2, h3 are used to provide section seperation.Paragraphs are written using the p tag which has the content of the page. Used unordered lists to implement key facts using the ul tag.Images are included usingbthe img tags to create a user friendly webpage.

CSS Techniques: The page follows an ocean themed design using background colors to fit that theme.The main section is centered using a max width of 1200px, and using margin auto for proper alignment.Line-height is used to improve readability of the page.The side bar made to stay in position when scrolling by using position: sticky;. The images are formatted to be fully responsive using max-width:100%,height: auto; and have rounded the edges uding border-radius.The links on the side bar has been linked to its respective section and has a hover effect of changing color when hovered.@media (max-width: 900px) is used to make the webpage adjustable to different screen sizes.CSS properties like display, flex, flex-direction, padding, margin,align-items is used to structure the page appropriately.

Accessibility

Ensured the image contains the alt attributes to support screen readers.Also ensured that there is a sufficient color contrast for readability. Ensured keyboard navigation with properly structured buttons ARIA labels are present for enhanced usability

Challenges and Lessons Learned

Splash-Screen: Challenges like ensuring the overlay enhanced the readability without affecting the background image,getting the content to align to the center including the animated loader appered when implementing this page.However these challenges were overcome by using CSS properties like flex box specially felx-direction: column to get the alignment of the entire content to the center of the page. The overlay was implemnted to be applied to the entire webpage by adding it the entire section. Also animations can be used to improve the user experience but should be lightweight.

Volunteer-page: Challenges like ensuring that the volunteer cards appear in a grid layout, maintaing a constant card height so all the content within the card would fit inside it clearly and displaying an error messsage when a feeback is typed and sent using the send feedback button were faced but thet were overcome by using CSS Grid efficiently to create the layout and using JavaScript to display the error messahe using and id. Also by balancing the designing with the funtionality to we can provide an engaing user experience.

Content-Page: Challenges like ensuring images maintain proper aspect ratios on all screen sizes and balancing content density with readability occured. However they were overcome by optimizing images using CSS techniques for responsive scaling. Maintaining accessibility best practices is important to provide the user an inclusive experience.

Compliance

The web pages created adhere to JANET web publication regulationshas.Has a sematic HTML structure using section, h1, p, button.Has alt text for images. Consistent CSS for readability and visual hierarchy. Has a responsive design with flexbox and grid for different screen size adaptation. Validated the HTML & CSS for comliance using W3C validation. .
